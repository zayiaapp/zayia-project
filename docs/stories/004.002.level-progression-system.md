## Problem Statement

Users complete challenges and earn points, but have no automatic level progression system:
- ❌ No automatic level-up detection
- ❌ No level-up bonus points
- ❌ No visual celebration/animation
- ❌ No level-up notifications
- ❌ Next level requirement not displayed

Currently: Points are awarded but level doesn't progress.

### Task 4.2.2: Implement Level-Up Detection Logic
**Owner:** @dev
**Effort:** 1.5 points

**Files:**
- Modify: `/src/lib/supabase-client.ts` (add checkAndAwardLevelUp function)

**Function:**
```typescript
async checkAndAwardLevelUp(
  userId: string,
  currentPoints: number,
  currentLevel: number
): Promise<{
  leveledUp: boolean
  newLevel: number
  bonusPoints: number
  totalBonus: number
}>
```

Logic:
1. Loop through LEVELS to find current level requirement
2. Check if `currentPoints >= nextLevelRequirement`
3. If yes: increment level, award bonus
4. If multiple level-ups: loop and accumulate bonuses
5. Update `profiles.level` in Supabase
6. Return level-up details
