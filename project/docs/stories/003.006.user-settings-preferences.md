## Problem Statement

Users have no way to customize their experience:
- ❌ No notification settings (email, in-app)
- ❌ No privacy settings
- ❌ Can't manage coaching preferences
- ❌ No data export/delete account option
- ❌ No dark mode/theme settings

**This story creates a settings page where users can control preferences, privacy, and account.**

### Task 6.1: Create user_preferences Table
**Owner:** @dev / @data-engineer
**Effort:** 1 point

**Migration:**
- Create `user_preferences` table with fields:
  - user_id (FK to profiles)
  - notifications_email (boolean)
  - notifications_inapp (boolean)
  - notifications_community (boolean)
  - profile_visibility (public/friends/private)
  - show_email (boolean)
  - show_last_access (boolean)

  - challenge_frequency (daily/weekly/monthly)
  - coaching_style (motivational/technical/balanced)
  - language (pt/en)
  - font_size (small/medium/large)
