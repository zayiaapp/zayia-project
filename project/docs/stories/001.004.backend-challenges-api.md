## Task Breakdown

### Task 4.1: Implement Challenge Completion
**Owner:** @dev
**Effort:** 2 points

**Functions to add:**
```typescript
completeChallenge(challengeId: string, userId: string, proofUrl: string): Promise<ChallengeCompletion>
```

**Validation:**
- Challenge exists
- User not banned
- Proof URL is valid (basic validation)
- User not already completed today

**Logic:**
- Calculate points: easy=10, medium=15, hard=25
- Create entry in user_progress table
- Call checkAndUnlockMedals() for medal logic
- Update user.points in profile
- Dispatch pointsUpdated event

**Response:**
```typescript
interface ChallengeCompletion {
  success: boolean
  points_earned: number
  medals_unlocked: string[]
  message: string
}
```
