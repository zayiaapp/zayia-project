# STORY 003.004: User Medals/Achievements Tab

**Status:** Ready for Dev 📋
**Complexity:** 6 points (MEDIUM)
**Created:** 2026-03-03
**Epic:** User Dashboard
**Depends On:** EPIC-001 ✅ + STORY-003.001

---

## Problem Statement

Users can't see their medal/badge achievements:
- ❌ No dedicated achievements view
- ❌ No way to see earned vs locked medals
- ❌ No progress toward earning medals (thresholds not shown)
- ❌ No medal details/rarity display

**This story creates a medals/achievements tab showing all earned badges and progress toward locked ones.**

---

## Acceptance Criteria

### AC1: Earned Medals Display
- [ ] Show all medals earned by user (from user_earned_badges)
- [ ] Display medal icon, name, description
- [ ] Show rarity level (color-coded: common/uncommon/rare/epic/legendary)
- [ ] Show when earned (date)
- [ ] Total count (X Medalhas)

### AC2: Locked Medals Progress
- [ ] Show locked medals with unlock requirements
- [ ] Display progress toward unlock (e.g., "Complete 3 of 5 challenges")
- [ ] Progress bar showing completion percentage
- [ ] Unlock condition tooltip on hover

### AC3: Medal Details Modal
- [ ] Click medal to see full details
- [ ] Large icon display
- [ ] Description and unlock condition
- [ ] Earned date (if earned) or unlock progress (if locked)
- [ ] Share button (optional: "I earned the X medal")

### AC4: Filter and Sort
- [ ] Filter by rarity (common, uncommon, rare, epic, legendary)
- [ ] Sort by earned date (newest first) or rarity
- [ ] Search by medal name
- [ ] Show count by rarity

### AC5: Real-time Updates
- [ ] Update when new medal earned
- [ ] Refresh on unlock triggers
- [ ] Listen to user_earned_badges changes

### AC6: Quality Gates Passed
- [ ] `npm run build` succeeds with 0 TypeScript errors
- [ ] `npm run lint` passes with 0 warnings
