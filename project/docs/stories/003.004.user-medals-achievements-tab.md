**Status:** Ready for Review ✅

**Epic:** User Dashboard
**Depends On:** EPIC-001 ✅ + STORY-003.001

## Problem Statement

Users can't see their medal/badge achievements:
- ❌ No dedicated achievements view
- ❌ No way to see earned vs locked medals
- ❌ No progress toward earning medals (thresholds not shown)
- ❌ No medal details/rarity display

**This story creates a medals/achievements tab showing all earned badges and progress toward locked ones.**

---

## Acceptance Criteria

### AC1: Earned Medals Display
- [x] Show all medals earned by user (from user_earned_badges)
- [x] Display medal icon, name, description
- [x] Show rarity level (color-coded: common/uncommon/rare/epic/legendary)
- [x] Show when earned (date)
- [x] Total count (X Medalhas)

### AC2: Locked Medals Progress
- [x] Show locked medals with unlock requirements
- [x] Display progress toward unlock (e.g., "Complete 3 of 5 challenges")
- [x] Progress bar showing completion percentage
- [x] Unlock condition tooltip on hover

### AC3: Medal Details Modal
- [x] Click medal to see full details
- [x] Large icon display
- [x] Description and unlock condition
- [x] Earned date (if earned) or unlock progress (if locked)
- [x] Share button (optional: "I earned the X medal")

### AC4: Filter and Sort
- [x] Filter by rarity (common, uncommon, rare, epic, legendary)
- [x] Sort by earned date (newest first) or rarity
- [x] Search by medal name
- [x] Show count by rarity

### AC5: Real-time Updates
- [x] Update when new medal earned
- [x] Refresh on unlock triggers
- [x] Listen to user_earned_badges changes
