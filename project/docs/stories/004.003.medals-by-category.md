# STORY-004.003: Medal/Badge System - By Category

**Status:** Draft ✍️
**Complexity:** 4 pontos
**Created:** 2026-03-04
**Epic:** EPIC-004: Gamification System
**Depends On:** STORY-001.006, STORY-004.001

## Problem Statement

Users earn medals but have no clear view of what medals exist in each category:
- ❌ No category-based medal organization
- ❌ Locked medals not displayed (no unlock requirements shown)
- ❌ No medal detail modal (rarity, requirements, reward points)
- ❌ No progress tracking (X/Y medals in category)
- ❌ No unlock progress indicator

Currently: Global medals exist but no category-based structure.

---

- [ ] List all medals organized by category (Saúde, Carreira, etc)
- [ ] Show earned medals with checkmark/animation

### AC2: Locked Medal Unlock Requirements ✅
- [ ] Display requirement for each locked medal

- [ ] Show progress toward unlock: "2/5 desafios"
- [ ] Update progress in real-time

- [ ] Click medal → open detail modal
- [ ] Show medal icon (large, 200x200px)
- [ ] Show name, category, rarity color
- [ ] Show description and requirements
