## Problem Statement

Após todas as stories de remoção de mock data, precisamos de uma auditoria formal que garante que **nenhum dado fake permanece no codebase**.

Sem esta auditoria, é possível que:
- Dados hardcoded foram esquecidos em componentes secundários
- Imports de arquivos de mock ainda existem mas não são usados (dead code)
- localStorage ainda é usado como source of truth em algum lugar
- Algum `Math.random()` ainda está gerando dados falsos

---

## Acceptance Criteria

### AC1: Script de auditoria — verificações automatizadas

- [ ] Criar `scripts/audit-mock-data.sh`:

```bash
#!/bin/bash
# Auditoria de mock data — deve retornar 0 ocorrências em tudo

echo "🔍 Auditoria de Mock Data — ZAYIA"
echo "=================================="
ERRORS=0

# 1. Imports de arquivos de mock
echo ""
echo "1️⃣  Imports de mock data files:"
MOCK_IMPORTS=$(grep -rn "import.*mock\|from.*mock" src/ --include="*.tsx" --include="*.ts" 2>/dev/null)
if [ -z "$MOCK_IMPORTS" ]; then
  echo "   ✅ Nenhum import de mock encontrado"
else
  echo "   ❌ Imports de mock encontrados:"
  echo "$MOCK_IMPORTS" | sed 's/^/   /'
  ERRORS=$((ERRORS + 1))
fi

# 2. generateMock functions
echo ""
echo "2️⃣  Funções generateMock:"
GEN_MOCK=$(grep -rn "generateMock\|MockData\|mockAll" src/ --include="*.tsx" --include="*.ts" 2>/dev/null)
if [ -z "$GEN_MOCK" ]; then
  echo "   ✅ Nenhuma função generateMock"
else
  echo "   ❌ generateMock encontrado:"
  echo "$GEN_MOCK" | sed 's/^/   /'
  ERRORS=$((ERRORS + 1))
fi

# 3. Math.random() nos componentes (exceto arquivos de teste)
echo ""
echo "3️⃣  Math.random() nos componentes:"
MATH_RANDOM=$(grep -rn "Math.random()" src/ --include="*.tsx" --include="*.ts" 2>/dev/null)
if [ -z "$MATH_RANDOM" ]; then
  echo "   ✅ Nenhum Math.random()"
else
  echo "   ❌ Math.random() encontrado:"
  echo "$MATH_RANDOM" | sed 's/^/   /'
  ERRORS=$((ERRORS + 1))
fi

# 4. Dados hardcoded suspeitos (nomes de pessoas em português)
echo ""
echo "4️⃣  Nomes hardcoded suspeitos:"
HARDCODED=$(grep -rn '"Maria\|"Ana \|"Julia\|"Helena\|"Larissa\|"Carlos\|"Pedro' src/ --include="*.tsx" 2>/dev/null | grep -v "placeholder\|test\|mock")
if [ -z "$HARDCODED" ]; then
  echo "   ✅ Nenhum nome hardcoded"
else
  echo "   ⚠️  Nomes hardcoded (verificar se são dados reais ou exemplos de placeholder):"
  echo "$HARDCODED" | sed 's/^/   /'
fi

# 5. localStorage como source of truth para dados de usuário
echo ""
echo "5️⃣  localStorage para dados críticos de usuário:"
LS_CRITICAL=$(grep -rn "localStorage.getItem('user_points')\|localStorage.getItem('earned_badges')\|localStorage.getItem('user_streak')\|localStorage.getItem('zayia_user_challenges'" src/ --include="*.tsx" --include="*.ts" 2>/dev/null)
if [ -z "$LS_CRITICAL" ]; then
  echo "   ✅ Nenhum localStorage crítico"
else
  echo "   ❌ localStorage para dados críticos:"
  echo "$LS_CRITICAL" | sed 's/^/   /'
  ERRORS=$((ERRORS + 1))
fi

# 6. seedLocalStorage (workaround que foi removido)
echo ""
echo "6️⃣  seedLocalStorage (workaround):"
SEED_LS=$(grep -rn "seedLocalStorage" src/ 2>/dev/null)
if [ -z "$SEED_LS" ]; then
  echo "   ✅ seedLocalStorage removido"
else
  echo "   ❌ seedLocalStorage ainda presente:"
  echo "$SEED_LS" | sed 's/^/   /'
  ERRORS=$((ERRORS + 1))
fi

# 7. Métricas hardcoded específicas (do CEO Dashboard)
echo ""
echo "7️⃣  Métricas hardcoded do CEO:"
CEO_METRICS=$(grep -rn '"1,247"\|"892"\|"156"\|"2,834"\|>1,247<\|>892<\|>156<\|>2,834<' src/ --include="*.tsx" 2>/dev/null)
if [ -z "$CEO_METRICS" ]; then
  echo "   ✅ Nenhuma métrica hardcoded do CEO"
else
  echo "   ❌ Métricas hardcoded:"
  echo "$CEO_METRICS" | sed 's/^/   /'
  ERRORS=$((ERRORS + 1))
fi

# 8. Resumo
echo ""
echo "=================================="
if [ $ERRORS -eq 0 ]; then
  echo "🎉 AUDITORIA PASSOU — Zero mock data!"
else
  echo "❌ AUDITORIA FALHOU — $ERRORS categorias com problemas"
  exit 1
fi
```

- [ ] Tornar executável: `chmod +x scripts/audit-mock-data.sh`
- [ ] Rodar: `bash scripts/audit-mock-data.sh`

### AC2: Arquivos de mock que podem ser deletados após auditoria

- [ ] Verificar com grep se cada arquivo ainda é usado. Se não for usado, deletar:

```bash
# Verificar se cada arquivo de mock ainda é importado
echo "📋 Verificando arquivos de mock:"
for file in \
  "src/lib/challenges-data-mock.ts" \
  "src/lib/badges-storage.ts" \
  "src/lib/challenges-storage.ts" \
  "src/lib/ranking-data-mock.ts"
do
  filename=$(basename "$file" .ts)
  USAGE=$(grep -rn "$filename" src/ --include="*.tsx" --include="*.ts" 2>/dev/null | grep "import")
  if [ -z "$USAGE" ]; then
    echo "✅ $file — pode ser deletado (não importado)"
  else
    echo "⚠️  $file — ainda importado:"
    echo "$USAGE"
  fi
done
```

- **Impacto:** TypeScript error se deletar antes de mover.
