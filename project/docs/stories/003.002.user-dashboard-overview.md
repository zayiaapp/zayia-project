# STORY 003.002: User Dashboard Overview

**Status:** Ready for Dev 📋
**Complexity:** 6 points (MEDIUM)
**Created:** 2026-03-03
**Epic:** User Dashboard
**Depends On:** EPIC-001 ✅ + STORY-003.001

---

## Problem Statement

Users have no clear summary of their progress:
- ❌ No central view of points, level, badges
- ❌ No quick stats (challenges completed, ranking position)
- ❌ No motivation widget showing progress
- ❌ Stats scattered across multiple sections

**This story creates a dashboard overview with real-time stats and progress visualization.**

---

## Acceptance Criteria

### AC1: User Stats Display
- [ ] Total points (from profiles.points)
- [ ] Current level (0-9, calculated from points)
- [ ] Badges earned count
- [ ] Challenges completed count
- [ ] Current ranking position (if ranked)

### AC2: Progress Cards
- [ ] Points progress card with visual bar
- [ ] Badges progress (X of Y collected)
- [ ] Challenges progress (X completed, Y remaining)
- [ ] Ranking position (if user is ranked)
- [ ] Color-coded by rarity/difficulty

### AC3: Real-time Updates
- [ ] Points update when challenge completed
- [ ] Badges update when earned
- [ ] Ranking updates after monthly calculation
- [ ] Use Supabase listeners where possible
- [ ] Fallback to polling if needed

### AC4: Motivation Elements
- [ ] Next level progress bar (X points to next level)
- [ ] Next achievement hint (e.g., "Complete 5 challenges for Iniciante")
- [ ] Recent achievements (last 3 badges earned)
- [ ] Encouraging messages based on progress

### AC5: Quality Gates Passed
- [ ] `npm run build` succeeds with 0 TypeScript errors
- [ ] `npm run lint` passes with 0 warnings
- [ ] Component tested with real data
