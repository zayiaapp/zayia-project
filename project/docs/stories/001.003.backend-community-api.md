## Task Breakdown

### Task 3.1: Implement Message CRUD Functions
**Owner:** @dev
**Effort:** 2 points

**Functions to add:**
```typescript
postMessage(userId: string, content: string): Promise<CommunityMessage>
getMessages(limit?: number, offset?: number): Promise<CommunityMessage[]>
deleteMessage(messageId: string): Promise<boolean>
```

**Validation:**
- Content not empty
- Content max 500 characters
- User must be authenticated
- User must not be banned

**Error Handling:**
- Handle RLS violations
- Handle user not found
- Handle message not found
- Return typed error responses

### Task 3.2: Implement Reaction Functions
**Owner:** @dev
**Effort:** 1 point

**Functions to add:**
```typescript
addReaction(messageId: string, userId: string, emoji: string): Promise<MessageReaction>
removeReaction(messageId: string, userId: string, emoji: string): Promise<boolean>
```

**Validation:**
- Valid emoji (5 common: 👍 ❤️ 😂 🔥 😮)

- User must exist
- Message must exist

**Error Handling:**
- Duplicate reaction → ignore
- Invalid emoji → error
- Message not found → error
