# STORY 003.005: User Community Participation

**Status:** Ready for Dev 📋
**Complexity:** 6 points (MEDIUM)
**Created:** 2026-03-03
**Epic:** User Dashboard
**Depends On:** EPIC-001 ✅ + STORY-003.001

---

## Problem Statement

Users can't participate in community:
- ❌ No way to post messages
- ❌ Can't see messages from others
- ❌ No ability to react (emoji) to messages
- ❌ Can't report inappropriate content
- ❌ No community guidelines visible

**This story creates a community section where users can post, view messages, react, and report.**

---

## Acceptance Criteria

### AC1: Message Feed
- [ ] Display all community messages (except deleted)
- [ ] Show user avatar, name, message content, timestamp
- [ ] Pagination or infinite scroll (50 per page)
- [ ] Show soft-deleted messages as "[Mensagem deletada]"
- [ ] Order by newest first

### AC2: Post Message Modal
- [ ] Modal with text area (max 2000 chars)
- [ ] Character counter
- [ ] Post button with validation (not empty)
- [ ] Success message after posting
- [ ] Error handling if post fails
- [ ] Close button/ESC to cancel

### AC3: Message Reactions
- [ ] Show emoji reaction count per message
- [ ] User can add emoji reaction (emoji picker or quick emojis)
- [ ] User can remove their own reaction
- [ ] Real-time updates of reaction counts
- [ ] Show which user reacted (tooltip on hover)

### AC4: Report Functionality
- [ ] Report button on each message (for admins to review)
- [ ] Report modal with reason selection:
  - disrespectful
  - inappropriate
  - spam
  - discrimination
  - privacy
  - other
- [ ] Optional description field
- [ ] Confirmation message after reporting

### AC5: Community Rules Display
- [ ] Show community rules in collapsible section
- [ ] Rules text from community_rules table
- [ ] Remind users before posting first time

### AC6: Ban Detection
- [ ] Check if current user is banned before allowing post
- [ ] Show "You are banned until DATE" message if banned
- [ ] Hide post button if user is banned
- [ ] Show ban reason (if available)

### AC7: Quality Gates Passed
- [ ] `npm run build` succeeds with 0 TypeScript errors
- [ ] `npm run lint` passes with 0 warnings
