# STORY 003.005: User Community Participation

**Status:** Ready for Review ✅
**Complexity:** 6 points (MEDIUM)
**Created:** 2026-03-03
**Epic:** User Dashboard
**Depends On:** EPIC-001 ✅ + STORY-003.001

## Problem Statement

Users can't participate in community:
- ❌ No way to post messages
- ❌ Can't see messages from others
- ❌ No ability to react (emoji) to messages
- ❌ Can't report inappropriate content
- ❌ No community guidelines visible

**This story creates a community section where users can post, view messages, react, and report.**

---

## Acceptance Criteria

### AC1: Message Feed
- [x] Display all community messages (except deleted)
- [x] Show user avatar, name, message content, timestamp
- [x] Pagination or infinite scroll (50 per page)
- [x] Show soft-deleted messages as "[Mensagem deletada]"
- [x] Order by newest first

### AC2: Post Message Modal
- [x] Modal with text area (max 2000 chars)
- [x] Character counter
- [x] Post button with validation (not empty)
- [x] Success message after posting
- [x] Error handling if post fails
- [x] Close button/ESC to cancel

### AC3: Message Reactions
- [x] Show emoji reaction count per message
- [x] User can add emoji reaction (emoji picker or quick emojis)
- [x] User can remove their own reaction
- [x] Real-time updates of reaction counts
- [x] Show which user reacted (tooltip on hover)

### AC4: Report Functionality
- [x] Report button on each message (for admins to review)
- [x] Report modal with reason selection:
  - disrespectful
  - inappropriate
  - spam
  - discrimination
  - privacy
  - other
- [x] Optional description field
- [x] Confirmation message after reporting

### AC5: Community Rules Display
- [x] Show community rules in collapsible section
- [x] Rules text from community_rules table
- [x] Remind users before posting first time

### AC6: Ban Detection
- [x] Check if current user is banned before allowing post
- [x] Show "You are banned until DATE" message if banned
- [x] Hide post button if user is banned
- [x] Show ban reason (if available)

### AC7: Quality Gates Passed

- [x] `npm run lint` passes with 0 warnings
