# STORY-004.004: Global Medals System

**Status:** Draft ✍️
**Complexity:** 3 pontos
**Created:** 2026-03-04
**Epic:** EPIC-004: Gamification System
**Depends On:** STORY-004.001, STORY-004.003

## Problem Statement

Users earn category-specific medals but have no global medals for special achievements:
- ❌ No global medals (Ovo, Guerreira, Mestre, etc)
- ❌ No cross-category achievement tracking
- ❌ No cumulative milestone tracking
- ❌ No special rarity badges
- ❌ No visual distinction for global vs category medals

Currently: Only category-specific medals exist.

---

### Task 4.4.2: Implement Global Medal Unlock Detection ✅
**Owner:** @dev
**Effort:** 1.5 points

**Files:**
- Modify: `/src/lib/supabase-client.ts`

**Function:** `checkAndAwardGlobalMedals(userId, earnedMedalIds, totalPoints)`
- Input: user ID, list of earned medals, current points
- Returns: {unlockedMedals: string[], newUnlocks: Medal[]}
- Logic:
  1. Count total earned medals
  2. Check each global medal requirement
  3. Insert unlocked medals into user_earned_badges
  4. Return new unlocks only (prevent duplicates)
  5. Dispatch event for each new unlock
