# STORY 003.001: User Profile Section

**Status:** In Progress 🔄
**Complexity:** 5 points (SMALL)
**Created:** 2026-03-03
**Epic:** User Dashboard
**Depends On:** EPIC-001 ✅ + EPIC-002 ✅

---

## Problem Statement

Users currently view their profile in a scattered way:
- ❌ No dedicated profile page
- ❌ Profile edit mixed with settings
- ❌ Avatar management missing
- ❌ Account info (join date, last access) not displayed
- ❌ Address/contact info read-only in database

**This story creates a complete user profile section with editable profile info and persistent address storage.**

---

## Acceptance Criteria

### AC1: Profile Header Display ✅
- [x] Show user avatar (circular, 7xl)
- [x] Display full name (editable)
- [x] Show email (read-only)
- [x] No "points" or "level" here (belongs in Dashboard)

- [x] Membro desde: DD/MM/YYYY (from created_at)
