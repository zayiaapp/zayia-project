## Problem Statement

EPIC-001 created the questions database. Now we need:

- ❌ No admin UI to view all 21 onboarding questions
- ❌ No way to create new questions
- ❌ No way to edit existing questions
- ❌ No way to delete questions
- ❌ No validation for question types
- ❌ No enforcement of single "final" question

**This story creates the admin Quiz tab with full question CRUD operations.**

**Type Definitions:**
```typescript
interface QuestionInput {
  question: string
  alternatives: Array<{ text: string, is_correct: boolean }>
  category: string
  difficulty?: 'facil' | 'medio' | 'dificil'
  is_final?: boolean
  is_active?: boolean
}

interface QuestionUpdate extends Partial<QuestionInput> {}
```

**Validation Rules:**
- Question: min 10 chars, max 500
- Alternative: min 2, max 5
- Final question: max 1 per category
- Question type: required

**Error Messages:**
- "Pergunta deve ter pelo menos 10 caracteres"
- "Mínimo 2 alternativas necessárias"
- "Máximo 5 alternativas permitidas"
- "Apenas 1 pergunta final permitida"
