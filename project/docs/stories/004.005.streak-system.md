## Problem Statement

Users complete daily challenges but have no incentive to maintain consistency:
- ❌ No streak tracking (consecutive days)
- ❌ No streak display in dashboard
- ❌ No streak bonuses or rewards
- ❌ No streak loss notification when broken
- ❌ No "on fire" visual indicator

Currently: Each challenge is isolated with no continuity tracking.

---

## Acceptance Criteria

### AC1: Streak Calculation ✅
- [ ] Track consecutive days of challenge completion
- [ ] Increment on challenge completion (if today != last completion date)
- [ ] Reset to 0 if no challenge for 24+ hours
- [ ] Store in profiles.streak field
- [ ] Update on each challenge completion

### AC2: Streak Display ✅
- [ ] Show current streak in Dashboard (prominent position)
- [ ] Display: "🔥 X dias em chamas!" (X days on fire)
- [ ] Show fire emoji animation/pulse effect
- [ ] Show streak start date: "Começou em DD/MM"
- [ ] Mobile responsive display

### AC3: Streak Bonuses ✅
- [ ] Award bonus points for maintaining streak
- [ ] Bonus formula: base_points + (streak_count * multiplier)
- [ ] Example: 10-day streak = +10% to challenge points
- [ ] Cap maximum bonus at streak_50 = +50%
- [ ] Display bonus in challenge completion toast

### AC4: Streak Milestones ✅
- [ ] Unlock special streak milestones (7, 14, 30, 50, 100 days)
- [ ] Show "🎯 Parabéns! 7 dias em chamas!" on milestone
- [ ] Award extra points/badge on milestone
- [ ] Store milestones in user_streak_milestones table
- [ ] Prevent duplicate rewards

### AC5: Streak Loss Notification ✅
- [ ] Check if streak broken (24+ hours without challenge)
- [ ] Show warning toast: "⚠️ Sua sequência foi interrompida!"
- [ ] Display previous streak count: "Você tinha X dias"
- [ ] Show encouragement: "Comece uma nova sequência hoje!"
- [ ] Reset streak to 0

### AC6: Real-time Updates ✅
- [ ] Update streak immediately on challenge completion
- [ ] Dispatch `streakUpdated` event
- [ ] Dashboard listens and updates display
- [ ] No manual refresh needed
- [ ] Works across tabs/devices

### AC7: Quality Gates ✅
- [ ] `npm run build` succeeds with 0 TypeScript errors
- [ ] `npm run lint` passes with 0 warnings
- [ ] Streak calculation verified
- [ ] No console errors
