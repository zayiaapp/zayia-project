## Problem Statement

EPIC-001 created the user management APIs. Now we need:

- ❌ No UI to view all users in admin panel
- ❌ No search/filter functionality for users
- ❌ No way to create new users via admin
- ❌ No way to view user details (profile, activity, payments)
- ❌ No way to edit user information
- ❌ No way to deactivate/delete users
- ❌ No email notifications on user actions

**This story creates the admin Guerreiras tab with full user CRUD operations.**

## Dev Notes

**Implementation Order:**
1. Add user CRUD functions to supabase-client.ts
2. Create modals (Create, View, Edit, Delete)
3. Enhance GuerreirasSection with search/filter/pagination
4. Add email integration via Resend
5. Set up real-time listener for user updates

**Type Definitions:**
```typescript
interface UserInput {
  email: string
  full_name: string
  phone?: string
  subscription_plan: 'basic' | 'premium' | 'vip'
}

interface UserUpdate {
  full_name?: string
  email?: string
  phone?: string
  subscription_plan?: 'basic' | 'premium' | 'vip'
}

interface FilterOptions {
  status?: 'active' | 'cancelled' | 'expired'
  plan?: 'basic' | 'premium' | 'vip'
  page: number
  limit: number
}
```

**Real-Time Listeners:**
- Listen to profiles INSERT for new users
- Listen to profiles UPDATE for edits
- Refresh table on changes

**Error Handling:**
- Failed user creation → show error toast
- Failed email → show warning but allow user creation
- Failed delete → show error toast
- Failed fetch → show empty state with retry
