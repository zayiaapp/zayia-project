# ZAYIA 💜

**Sua mentora pessoal de transformação e crescimento**

ZAYIA é uma plataforma inovadora de coaching pessoal com IA, focada no empoderamento feminino e desenvolvimento pessoal.

## ✨ Características

### 🎨 Design Elegante
- **Paleta roxa/violeta** calmante e sofisticada
- **Interface responsiva** para mobile e desktop
- **Micro-interações** e animações suaves
- **Gradientes únicos** e elementos visuais marcantes

### 👑 Painel CEO
- **Dashboard executivo** com métricas em tempo real
- **Gestão de usuárias** e conversas
- **Analytics detalhado** e relatórios
- **Status do sistema** e monitoramento

### 💜 Experiência do Usuário
- **Chat inteligente** com respostas personalizadas
- **Acompanhamento de progresso** (streak, metas, sessões)
- **Interface acolhedora** e feminina
- **Gamificação** com níveis e conquistas

### 🔐 Autenticação Completa
- **Login/cadastro** com validação
- **Perfis diferenciados** (CEO/Usuário)
- **Acesso rápido CEO** para demonstrações
- **Segurança** com Supabase

## 🚀 Tecnologias

- **React 18** + TypeScript
- **Tailwind CSS** para styling
- **Supabase** para backend
- **Lucide React** para ícones
- **Vite** para desenvolvimento

## 🛠️ Configuração

1. **Clone o repositório**
2. **Instale as dependências:** `npm install`
3. **Configure o Supabase:** Clique no botão "Connect to Supabase" no topo
4. **Inicie o servidor:** `npm run dev`
